Proyecto Final de Coder House (v.1)

Este Proyecto consta de una web donde hay 3 modulos: usuario, posteo y moderador.
usuario tiene como caracteristicas, Nombre, Mail y Fecha de Nacimiento
posteo tiene como caracteristicas, Titulo y Cuerpo
moderador tiene como caracteristicas Nombre, Mail y Sector

cada modulo tiene una seccion donde se pueden ver los integrantes, otra donde se pueden buscar y otra donde crearlos.

Esta app actualmente solo tiene como funcion crear y ver los diferentes modulos.
